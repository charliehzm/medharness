# Contributor Covenant Code of Conduct · 行为准则

## 我们的承诺

为营造一个开放友好的环境，作为贡献者和维护者，我们承诺让参与本项目和社区的每个人都不受骚扰，无论年龄、体型、残疾、族裔、性征、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性身份与取向。

## 我们的标准

正面行为：
- 用友善包容的语言
- 尊重不同观点和经历
- 优雅地接受建设性批评
- 关注社区利益
- 对其他社区成员表示同理心

负面行为：
- 使用性化语言或图像、不受欢迎的性关注
- 钓鱼、侮辱 / 贬低评论、人身或政治攻击
- 公开或私下骚扰
- 未经许可公开他人隐私信息
- 其他在职业环境中合理认为不当的行为

## 我们的责任

项目维护者负责澄清可接受行为的标准，并对任何不当行为采取适当公正的纠正措施。

维护者有权且有责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、issue 和其他贡献，或临时 / 永久禁止其认为有不当、威胁、攻击或有害行为的贡献者。

## 适用范围

本行为准则适用于项目空间和公共空间内当个人代表项目或其社区时的行为。

## 强制执行

辱骂、骚扰或其他不可接受行为，可通过 `conduct@medharness.io` 联系项目团队报告。
所有投诉都会被审查和调查，并以维护者认为必要和适当的方式响应。

维护者有义务为事件的报告者保密。具体强制执行政策的更多细节可能会单独发布。

不善意遵守或执行行为准则的维护者，可能会面临项目其他领导成员决定的临时或永久后果。

## 归属

本行为准则改编自 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)。
